Welcome to Inception Project

A Docker Infrastructure

This project demonstrates a complete containerized infrastructure using Docker Compose. It includes NGINX with SSL/TLS, WordPress with PHP-FPM, MariaDB database, and additional bonus services like Adminer and a static website.


Explore Services

Core Services

NGINX Proxy

Reverse proxy server with SSL/TLS encryption. Handles all incoming requests and routes them to appropriate services securely.

Running on port 443

WordPress

Content management system with PHP-FPM. Fully configured WordPress installation with custom themes and plugins support.

PHP-FPM Ready

MariaDB

High-performance database server. Configured with persistent volumes and optimized for WordPress integration with remote access.

Database Active

System Architecture

Network Configuration

Custom Docker bridge network enabling secure inter-service communication. All services are isolated and communicate through defined interfaces.

Volume Management

Persistent data storage using Docker volumes. WordPress files and MariaDB data are preserved across container restarts and updates.

Security Features

SSL/TLS encryption, secure container networking, and isolated service environments. Each service runs with minimal required privileges.
